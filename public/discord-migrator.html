Tags extras que serão adicionadas junto com "discord-migration". Separe por vírgulas.

ℹ️ Como funciona:

  1. O sistema busca todas as mensagens do canal/thread de origem usando o bot token
  2. Identifica todos os arquivos (imagens, vídeos, anexos) nas mensagens e embeds
  3. Baixa cada arquivo do CDN do Discord
  4. Faz upload para o storage local
  5. Reposta as mensagens no canal de destino via webhook com os novos links
  6. Mantém o nome de usuário e avatar originais

📋 Para Fóruns/Threads:

  • Origem:
    • ID do Canal: O ID do canal de fórum
    • ID da Thread de Origem: O ID da postagem específica
  • Destino:
    • Webhook URL: Webhook do canal que contém a thread
    • ID da Thread de Destino: O ID da thread onde postar
  • O bot precisa ter acesso às threads

Atenção: O processo pode levar vários minutos dependendo da quantidade de mensagens e arquivos. O bot precisa ter permissão de leitura no canal/thread de origem.
